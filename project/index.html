📁 Proyek Saya

🧥 Aplikasi Penjualan Barang Bekas (2025)

Saya mengembangkan Aplikasi Penjualan Barang Bekas, sebuah marketplace web sederhana untuk menjual produk fashion bekas seperti pakaian, sepatu, jaket, dan celana yang masih layak pakai. Dibuat dengan Laravel 10, aplikasi ini memiliki dua peran utama: penjual dan pembeli, dengan tampilan bersih dan mudah digunakan. Proyek ini bertujuan sebagai solusi digital jual-beli yang ringan, sekaligus menjadi media latihan bagi saya dalam membangun fitur CRUD, manajemen stok, session, UI/UX, serta penerapan Laravel dalam studi kasus nyata.

📚 Aplikasi Perpustakaan Digital (2024)

Saya telah mengembangkan sebuah aplikasi web Perpustakaan Digital menggunakan Laravel 10, dilengkapi dengan fitur CRUD (Create, Read, Update, Delete) untuk memudahkan pengelolaan data. Aplikasi ini dirancang khusus untuk memenuhi kebutuhan siswa dan siswi, sehingga mereka dapat dengan mudah mengakses, meminjam, dan mengelola informasi buku secara efisien dan terstruktur.

🥬 Aplikasi Toko Sayur Lontong (2024)

Saya membuat Aplikasi Toko Sayur Lontong pada tahun 2024 menggunakan Laravel 10. Aplikasi ini dilengkapi dengan fitur CRUD dan dirancang untuk memudahkan penjual dalam mengelola dan menjual produknya, serta mempermudah pembeli dalam melakukan pembelian secara praktis dan efisien.
